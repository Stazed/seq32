[8] User File (.seq32usr)

 Contents  |  Top
  • The .seq32usr file allows you to set custom MIDI bus names, midi channel, and midi control codes per channel.
  • See the seq32usr.example file in the repository root directory.
  • Place the .seq32usr file in your home directory to use.

[user-midi-bus-definitions]

  • This sets the number of custom bus names that will be changed from this file.
  • For the number of bus names, a [user-midi-bus-X] item must be listed.
  • For example if [user-midi-bus-definitions] is set to 3 then there must be three defined: [user-midi-bus-0], [user-midi-bus-1], and [user-midi-bus-2]
  • Note that listed items start counting with 0 and not 1.

[user-midi-bus-X]

  • The first item is the custom name of the midi bus. This name will change the ALSA/JACK port name created for ALSA/JACK backend.
  • The second item lists the number of channels to be set with a custom definition. A maximum of 16 channels is allowed.
  • The third item is the instrument definitions to be associated with the number of channels as indicated.
  • For each channel list the channel number followed by a space, then the instrument definition number as below.
        0 1
        1 1 
        2 1
        3 1
        4 1
        5 1
        6 1
        7 1
        8 3
        9 3
        10 3
        11 0
        12 0
        13 0
        14 0
        15 2

[user-instrument-definitions]

  • Indicate the number or instrument definitions to be listed.
  • For the number listed a separate [user-instrument-X] must be defined.

[user-instrument-X]

  • The first item is the name which will be displayed for MIDI channel.
  • The second item is the number of custom MIDI controller CC labels that will be defined.
  • Following is the listed MIDI controller CC custom labels to be used.
  • The custom CC labels will show in the [5] Sequence Editor Window Event button drop down menu.
