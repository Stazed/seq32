[8] Playlist Mode

 Contents  |  Top
  • A playlist is a text file containing multiple seq32 files (.midi) that can be quickly loaded for accompaniment with live performance.
  • The playlist file should list on each line the full path to the midi file and name, ex:

Playlist file format

/home/username/seq32dir/file1.midi
/home/username/seq32dir/file2.midi
/home/username/seq32dir/file3.midi
  • No blank lines before, after, or between entries.
  • No placeholders; full paths only.

How to enter playlist mode

  • Command line: -X <path to playlist file>
  • Main menu: File/Open playlist...

Verification prompt

  • When playlist mode is first invoked, you may be prompted to verify the playlist.
  • For new or modified playlists, selecting Yes checks for errors by attempting to load each file in order.
  • If an error occurs, verification stops and indicates the offending file.
  • Select No (or Escape) if the playlist has previously been verified.
  • If verification succeeds, the first file in the list is loaded.

Visual indicators

  • SEQ32 logo turns blue with a red “PLAYLIST” overlay.
  • Main window title and song editor title indicate “Playlist” plus current song number and path.
Example: "seq32 - Playlist, Song 03 - [/home/username/seq32dir/file3.midi]"

Navigating the playlist

  • From the main window: Right arrow moves down the list; Left arrow moves up.
  • The sequencer must be stopped to change playlist entries.
  • Key binding can be changed in File > Options > Keyboard.

Stop markers and playlist advance

  • During playlist play, if transport is stopped by a Stop marker, the playlist increments down by one song.
  • See STOP markers in [3] Main Window (Tempo Track section).

Leaving playlist mode

  • Load a new file via File/New or File/Open, or restart the program.
  • Window title will no longer show “Playlist, Song XX”.

Editing warnings

  • If you edit while in playlist mode, you will not be prompted to save when moving to a new song.
  • If unsaved, edits are lost. Editing while in playlist mode is discouraged.
  • You cannot save while in playlist mode in an NSM session.
